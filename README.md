# vue3-leaflet-map-component

该项目为vue3+leaflet组件，欢迎一起加入继续完善.

## 安装(install)

npm install vue3-leaflet-map-component

## 使用(use)

```
import { LsMap, LsMapDivMarker, LsMapGeo, LsMapMarker, LsMapPopup } from 'vue3-leaflet-map-component';
```
